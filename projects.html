I had done my project on ATM Card Security

PROJECT NAME: "ATM CARD SECURITY WITH RFID AND (one time password)OTP THROUGH GSM"

Why i Choose this project?

        

        1.The Proposed System provides the security for the ATM with RFID and one time password(OTP) through GSM.
        2.In the case when the card is stolen, or pin is known to unauthorized person, we require a high level of security which coined up on an idea of adding OTP and RFID to the current technology.
        3.OTP has emerged as a major for highly secure identification and personal verification
